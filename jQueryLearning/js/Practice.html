What is jQuery?

jQuery is a fast, small, and feature-rich JavaScript library. It makes things like HTML document traversal and manipulation, event handling, animation, and Ajax much simpler with an easy-to-use API that works across a multitude of browsers. With a combination of versatility and extensibility, jQuery has changed the way that millions of people write JavaScript.

Welcome To jQuery World

Advantages: Wide range of plug-ins that allows developers to create plug-ins on top of the JavaScript library. Large development community. It is a lot easier to use compared to standard javascript and other javascript libraries. It lets users develop Ajax templates with ease. Ajax enables a sleeker interface where actions can be performed on pages without requiring the entire page to be reloaded.
